# Screenshots

Use this folder for GitHub and portfolio screenshots.

1. Run the backend:

```bash
npm run dev:backend
```

2. Run the frontend:

```bash
npm run dev:frontend
```

3. Open:

```text
http://localhost:5173
```

4. Capture:

- `dashboard.png`
- `hero-detail.png`

5. Save them here:

```text
docs/screenshots/dashboard.png
docs/screenshots/hero-detail.png
```
